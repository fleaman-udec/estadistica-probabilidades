Análisis bivariado¶

Concepto de análisis bivariado¶

El anlisis bivariado estudia las relaciones empricas entre dos variables estadsticas

La idea es descubrir asociaciones entre las variables y su grado de asociacin

Covarianza¶

As como la varianza indica el grado de variabilidad de una variable estadstica, la covarianza indica el grado de variabilidad conjunta de dos variables estadsticas.

Si valores grandes (pequeos) de una variable se corresponden con valores grandes (pequeos) de la otra variable, entonces la covarianza es positiva.

En caso contrario la covarianza resulta ser negativa. Es decir, valores pequeos se corresponden con valores grandes y viceversa.

Matemticamente la covarianza muestral entre dos variables $X$ e $Y$ se estima de la siguiente forma:

\begin{align} \text{Cov}(X, Y) = \sum_{i=1}^{n} \frac{ (x_{i} - \bar{x}) (y_{i} - \bar{y}) }{n-1} \end{align}

Notar que la variaza muestral es un caso particular de la covarianza muestral cuando las dos variables son idnticas

Medidas de Correlación¶

Coeficiente de correlación (Pearson)¶

La normalizacin de la covarianza muestral resulta en el coeficiente de correlacin de Pearson, que es una medida normalizada de la correlacin lineal entre dos variables.

Matemticamente el coeficiente de correlacin (Pearson) se calcula de la siguiente forma para una muestra:

\begin{align} r_{xy} = \frac{1}{n-1} \sum_{i=1}^{n} \frac{ (x_{i} - \bar{x}) (y_{i} - \bar{y}) }{S_{x} S_{y} } \end{align}

El valor de $r$ puede variar entre -1 y 1.

Un $r$ mayor a cero indica una correlacin lineal positiva, mientras que un $r$ menor a cero indica una correlacin lineal negativa.

Un $r$ cercano a cero indica que no hay correlacin lineal.

Ejemplo En una minera se quiere estudiar la relacin entre la densidad del mineral y la potencia del molino SAG para alcanzar una molienda ptima.

import numpy as np
import matplotlib.pyplot as plt
np.random.seed(2)
density = 2100*(1+(np.random.randn(100)-0.5)/10)
power1 = (1+(np.random.randn(100)-0.5)/8)*density*130/700
power2 = (1+(np.random.randn(100)-0.1)/20)*density*130/700

fig, ax = plt.subplots(ncols=2)
fig.set_size_inches(16, 4.5)
ax[0].scatter(density, power1), ax[1].scatter(density, power2)
ax[0].set_xlabel('Densidad [kg/m3]'), ax[0].set_ylabel('Potencia [kW]'), ax[0].set_title('r = ' + str(np.corrcoef(density, power1)[0][1]))
ax[1].set_xlabel('Densidad [kg/m3]'), ax[1].set_ylabel('Potencia [kW]'), ax[1].set_title('r = ' + str(np.corrcoef(density, power2)[0][1]))
plt.show()

Diferencias con coeficiente de determinación¶

El coeficiente de correlacin $r$ es una medida de la asociacin lineal entre dos variables.

El coeficiente de determinacin $R^{2}$ es una medida de qu tan bien un modelo de regresin lineal puede ajustar a dos variables.

Obviamente dos variables que tengan una alta linealidad entre ellas, tendrn ms chances de poder modelarse correctamente mediante una regresin lineal.

Coeficiente de correlación (Spearman)¶

Se trata de un coeficiente de correlacin de rango entre dos variables aleatorias.

Bsicamente mide qu tan bien la relacin entre dos variables puede describirse mediante una funcin monotnica.

Correlación y causalidad¶

Es importante notar que una correlacin es cualquier relacin estadstica causal o no entre dos variables aleatorias.

Esto significa que a pesar que tengan una alta correlacin, no significa que uno de los fenmenos sea la causa del otro.

La correlacin es una condicin necesaria, pero no suficiente para establecer causalidad.

Sumario¶

  • El anlisis bivariado estudia las relaciones estadsticas entre dos variables.
  • El coeficiente de correlacin (Pearson) cuantifica el grado de relacin lineal entre dos variables.
  • Que dos variables tengan una alta correlacin no implica necesariamente que tengan una relacin causa-efecto.
